# Spazeo Brand Guidelines
> Version 1.0 — February 2026 | "Step Inside Any Space"

---

## 1. Brand Foundation

### Mission
Make every physical space explorable from anywhere, powered by AI and immersive 360° technology.

### Positioning
For real estate professionals and property managers who need to showcase spaces remotely,
**Spazeo** is the AI-powered 360° virtual tour platform that turns a single panorama into an
immersive, walkable experience — powered by Gaussian Splatting, AI staging, and depth estimation.

### Brand Pillars
| Pillar | Meaning |
|---|---|
| **Immersive** | Every interaction should feel like stepping into a real space |
| **Intelligent** | AI works behind the scenes — creation is effortless |
| **Accessible** | Anyone can create and explore — no 3D expertise needed |

### Target Audience
- Real estate agents and brokerages differentiating listings
- Property management companies showcasing rentals at scale
- Architects and interior designers presenting to clients
- Hotels, resorts, and venues offering virtual previews
- Commercial real estate firms marketing office and retail space

## 2. Name & Pronunciation

- **Origin:** Space → Spazeo (inspired by Italian suffix "-eo", premium global feel)
- **Pronunciation:** spa·ZAY·oh `/spəˈzeɪ.oʊ/`
- **In running text:** `Spazeo` (title case — never SPAZEO or spazeo)
- **In logo lockup:** `SPAZEO` (all caps with letter-spacing)
- **Never:** split across lines, hyphenate, or add suffixes (no "SpazeoApp", "Spazeo.io")
- **In a sentence:** "Built with Spazeo" or "Powered by Spazeo"

### Primary Tagline
> **"Step Inside Any Space"**
4 words. Benefit-driven. Works across all marketing contexts. Keep all taglines under 6 words.

### Alternative Taglines
- "See it before you're there" — benefit statement
- "Why show photos when you can show presence?" — challenge format
- "From panorama to walkthrough in seconds" — promise format
- "The anti-slideshow" — category twist

## 3. Brand Voice

### We Are / We Are NOT
| We Are | We Are NOT |
|---|---|
| Confident — our tech is powerful | Arrogant — we never talk down |
| Clear & Direct — plain language | Oversimplified — we respect intelligence |
| Innovative — we lead with new tech | Jargon-heavy — no buzzword salads |
| Warm & Inviting — like opening a door | Overly casual — no slang or emojis in copy |
| Empowering — users feel capable | Patronizing — never condescending |

### Good Copy Examples
- "Your tour is ready — share it with a single link."
- "We handle the complex 3D processing so you don't have to."
- "Upload a panorama. Get an immersive tour. It's that simple."
- "Your buyers can walk through properties from their couch."

### Bad Copy Examples (avoid)
- "Leveraging our proprietary neural pipeline to synthesize volumetric representations..." — jargon
- "hey!! your tour is done lol click here" — too casual
- "Obviously, anyone can see this is the best solution." — arrogant
- "Don't worry, it's super easy even for non-tech people!" — patronizing

### Personality Keywords
**USE:** Immersive · Intelligent · Effortless · Premium · Trustworthy · Seamless · Walkable

**AVOID:** Complicated · Cheap · Generic · Cold · Hype-driven · Disruptive · Synergy

## 4. Logo

### Wordmark Specification
The Spazeo logo is a wordmark in **Plus Jakarta Sans**, extra black weight (900),
with tight tracking (-0.5px). A small gold dot follows the final letter.

```
SPAZEO.
       ^-- Gold dot (#D4A017), 6px diameter
```

**Logo colors:**
- Text: `#F5F3EF` on dark backgrounds
- Dot: `#D4A017` (Brand Gold) — always gold regardless of context

### Versions
| Version | Usage |
|---|---|
| Full lockup (horizontal) | Primary usage — wordmark + dot |
| Symbol only | Favicons, app icons, small spaces |
| Stacked | Square formats — symbol above wordmark |

### Specifications
- Minimum width: **80px** (digital) / **20mm** (print)
- Clear space: equal to height of the letter "S" on all sides
- Primary color: `#F5F3EF` wordmark + `#D4A017` dot on dark backgrounds
- File formats: SVG (primary), PNG (transparent), PDF (print)

### Logo Rules
| DO | DON'T |
|---|---|
| Use official files only — never recreate | Stretch, skew, or rotate |
| Maintain clear space always | Add drop shadows or outer glows |
| Use on solid or simple gradient backgrounds | Place on busy photographic backgrounds |
| Scale proportionally | Change colors outside approved palette |
| | Add outlines or strokes |
| | Use below 80px width |

```html
<!-- On a single section -->
<section class="light">...</section>

<!-- On the entire page -->
<html data-theme="light">...</html>

<!-- Via JavaScript toggle -->

```

All CSS variables automatically flip when `.light` or `[data-theme="light"]` is applied.
Utility classes (`.glass`, `.shadow-glow`, `.bento-card`, `.bg-gradient-hero`, `.orb-*`, `.text-gradient`) also adapt.

## 6. Typography

### Typefaces
| Font | Role | CSS Variable |
|---|---|---|
| **Plus Jakarta Sans** | Display headings, hero text | `--font-jakarta` |
| **DM Sans** | Body text, UI, labels | `--font-dmsans` |

Both loaded as Google variable fonts with `display: swap`.

### Type Scale
| Element | Size / Line-height | Weight | Font |
|---|---|---|---|
| Display | 48–78px / tight | Black (900) | Plus Jakarta Sans |
| H1 | 36px / 40px | ExtraBold (800) | Plus Jakarta Sans |
| H2 | 28–52px / tight | Bold (700) | Plus Jakarta Sans |
| H3 | 22px / 28px | Semibold (600) | Plus Jakarta Sans |
| Body L | 18px / 28px | Regular (400) | DM Sans |
| Body | 16px / 24px | Regular (400) | DM Sans |
| Small | 14px / 20px | Regular/Medium | DM Sans |
| Caption | 12px / 16px | Medium (500) | DM Sans |

### Typography Rules
- Line length: **60–80 characters** max
- Never skip heading levels (H1 → H2 → H3)
- Letter spacing: -2px for display, -1.5px for H2, 0.01em for body
- Plus Jakarta Sans: headings only — never body text
- DM Sans: body/UI only — never hero headings
- ALL CAPS: only in labels, badges, and logo wordmark

## 12. Accessibility

- Gold `#D4A017` on `#0A0908` = **8.2:1** contrast
- Text `#F5F3EF` on `#0A0908` = **17.8:1** contrast
- All interactive elements: `focus-visible` outline (2px solid `#D4A017`)
- Never use red/green alone to convey meaning
- All images require `alt` text
- WCAG AA minimum: 4.5:1 body, 3:1 large text

## 13. Quick Reference — DO / DON'T

| DO | DON'T |
|---|---|
| Use dark backgrounds everywhere | Use light/white backgrounds |
| Gold for primary actions and accents | Purple, blue, or other off-brand colors |
| Teal for spatial/navigation features | Mix gold and teal in the same element |
| Coral sparingly for urgent CTAs | Overuse coral — it's 10% max |
| Plus Jakarta Sans for headings | Decorative or script fonts |
| DM Sans for body text | Use heading fonts for body |
| 4px spacing grid | Eyeball spacing |
| Keep taglines under 6 words | Verbose taglines |
| Test all colors for WCAG AA | Use low-contrast combinations |
| Write short, confident sentences | Use jargon or buzzwords |
